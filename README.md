# CodeBox Frontend — Online Coding Platform

## 🚀 Live Demo
https://codebox-frontend.vercel.app

## 🛠️ Tech Stack
- Next.js + TypeScript
- Tailwind CSS
- Redux Toolkit
- JWT Authentication
- Axios

## ✨ Features
- User authentication
- Online code editor
- Real-time collaboration

## ⚙️ Setup Locally
- Clone the repo
- Run `npm install`
- Create `.env.local` file
- Run `npm run dev`
